# Banking-Analytics-Aashboard-PowerBI
## Project Overview
This project focuses on analyzing banking data to understand customer distribution, 
account balances, transaction activity, loan exposure, and branch-level performance.
The dashboard helps identify financial risks and operational insights to support 
data-driven decision-making.

## Tools & Technologies
- Power BI
- DAX

## Dashboard Coverage
- Customer & account overview
- Transaction analysis (deposits vs withdrawals)
- Loan portfolio and loan status analysis
- Loan type distribution
- Branch-level performance comparison

## Key Business Insights
- High outstanding loan exposure compared to received payments
- Majority of loans are active, indicating ongoing credit risk
- Home loans dominate the loan portfolio
- Savings accounts contribute more balance than current accounts
- Transaction flow is stable with balanced deposits and withdrawals
- Significant performance variation across branches
